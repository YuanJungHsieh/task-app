# task-manager

### 功能：
- CRUD 事件/ 使用者
- 驗證使用者
- 創建或刪除使用者時，發送關心email
- 上傳使用者照片

### 環境 & 使用工具
- Node.js
- Express
- MongoDB / Mongoose
- npm
  - ejs
  - request
  - jsonwebtoken
  - dotenv
  - multer
  - bcrypt
  - validator
  - sharp
- SendGrid
